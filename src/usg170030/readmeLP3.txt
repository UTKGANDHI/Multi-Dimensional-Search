 Steps for running the LP3Driver.java file:

1. Compile and Run the LP3Driver.java file using the below commands:
		> javac LP3Driver.java
		> java LP3Driver 
				OR 
		> java LP3Driver <input-file-path>
		
		
2. Alternatively it can be run in Intellij/ Eclipse or any Java Compatible IDE.
